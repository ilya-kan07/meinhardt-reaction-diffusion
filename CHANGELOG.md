# CHANGELOG

## v1.1.0 (2025-12-21)

### Added
- Новая вкладка **История** — просмотр всех сохранённых расчётов
- Поддержка **SQLite базы данных** (`meinhardt_results.db`) для хранения результатов
- Сохранение всех слоёв при m ≤ 10000
- Умное прореживание слоёв при больших m (ранние + поздние + равномерно по середине)
- Возможность удаления расчётов из истории
- Автоматическое создание БД рядом с EXE в собранной версии

### Improved
- Полная совместимость с PyInstaller: БД и ресурсы работают корректно в автономном EXE
- Улучшена стабильность и портативность приложения
- Улучшена точность расчетов

---

## v1.0.1 (2025-11-15)

### Fixed
- `MeinhardtSolver.exe` теперь **полностью автономный** — не требует папку `resources/` рядом с EXE.
- Ресурсы (`conditions.png`, `main_conditions.png`, `solver.ico`) включены в бандл через PyInstaller.
- Исправлена ошибка `FileNotFoundError` при запуске EXE в произвольной директории.

### Improved
- Улучшена сборка: ресурсы корректно определяются через `Path.cwd()` в `.spec` файле.
- Код `get_base_path()` теперь надёжно работает как в режиме разработки, так и в собранном EXE.

---

## v1.0.0 (2025-04-05)

### Added
- Первая стабильная версия
- Две независимые программы:
  - **Численное решение** (`run_solver.py`)
  - **Аналитический анализ** (`run_analysis.py`)
- GUI на Tkinter с вкладками:
  - Начальные условия, численное решение, подбор параметров, метод Ньютона, равновесия
- 2D визуализация с анимацией слоёв
- Явная конечно-разностная схема с контролем устойчивости (условие Куранта)
- Unit-тесты + CI/CD (GitHub Actions)
- Сборка в `.exe` через PyInstaller
- Поддержка экспорта результатов в `.csv` и графиков

### Documentation
- `README.md` — обзор проекта
- `USER_GUIDE.md` — инструкция для пользователей
- `DEVELOPER_GUIDE.md` — для разработчиков
- `REPORT.pdf` — научный отчёт
